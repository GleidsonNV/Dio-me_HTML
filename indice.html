O que é HTML?

A linguagem de marcação de texto ou HTML é a estrutura das páginas web que navegamos diariamente na web. A forma que ela é entendida é por meio de tags
.

Tags

HTML
são como abre e fecha de parênteses em linguagens de programação normais. São iniciadas por símbolo menor que um nome e fechado por um maior que. Assim é possível saber como apresentar e identificar o conteúdo dentro das tags.

Cliente

O Cliente é justamente o que está sendo ensinado na dio.me com o trio must know html, css e javascript. É a apresentação da página e meio de comunicação com o servidor por parte do usuários.

Apesar do javascrip ser usado aqui como forma de adicionar interatividade às páginas web, com uso de diversos frameworks ele pode ser usado para aplicações desktop e muitos outros destinos.

Servidor

Este é o lado dos dados das páginas na internet. Aqui é guardado login e senha, por exemplo, e muitos outros conteúdos que serão apresentados ao usuário em seu navegador conforme interação.

Conclusão

Assim, vimos que páginas html são criadas e entendidas por meio das tags, onde é estruturado o website e apresentado ao usuário final. Algumas tecnologias importantes acrescentadas ao HTML são:

      CSS

      Javascript
